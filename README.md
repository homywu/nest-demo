# Nest.js Demo

## Description

Please use watch mode spin up the service, to view the Swagger docs, here is the link: <http://localhost:3000/docs/>

## Installation

```bash
yarn
```

## Running the app

```bash
# development
yarn start

# watch mode
yarn start:dev

# production mode
yarn start:prod
```

## Test

```bash
# unit tests
yarn test

# e2e tests
yarn test:e2e

# test coverage
yarn test:cov
```
